
# go-graphql

Sample project to understand GraphQL implementation in Golang

## Run

- start server

    ```shell script
    make run-server
    ```

- (a) start client

    ```shell script
    make run-client
    ```

or

- (b) start Graph*i*QL
    
    ```shell script
    open http://localhost:8080/query
    ```
    
    and copy paste queries and mutations from [examples](graphiql.md)
